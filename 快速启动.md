# Remote Agent Console - 快速启动指南

## 🚀 一键启动（推荐）

### 首次使用

```bash
cd dev
pnpm run setup
```

这个命令会自动：
- 检查环境
- 创建 .env 配置文件
- 安装所有依赖
- 构建项目

### 启动服务

```bash
pnpm start
```

或者

```bash
pnpm dev
```

服务会在独立窗口中启动，访问：
- **Web 控制台**: http://localhost:5173
- **Host API**: http://localhost:3001

### 登录账号

- 用户名: `admin`
- 密码: 首次启动时 Host 会自动生成并打印到控制台

如需固定密码，请在 `.env` 中设置 `ADMIN_PASSWORD`

---

## 📝 详细说明

### setup 命令
- 首次使用时运行一次即可
- 会自动配置环境和构建项目
- 如果已经运行过，可以跳过

### start 命令
- 在独立窗口中启动 Host 和 Web 服务
- 关闭窗口即可停止服务
- 适合日常使用

### dev 命令
- 在当前窗口启动服务（显示日志）
- 按 Ctrl+C 停止所有服务
- 适合开发调试

---

## 🔧 手动启动（高级）

如果需要分别启动服务：

### 启动 Host
```bash
pnpm run dev:host
```

注意查看控制台输出的临时密码（如果未在 .env 中配置）

### 启动 Web（新开终端）
```bash
pnpm run dev:web
```

---

## ❓ 常见问题

### 提示未找到 pnpm
```bash
npm install -g pnpm
```

### 首次启动失败
确保先运行了 `pnpm run setup`

### 端口被占用
修改 `.env` 文件中的 `HOST_PORT` 和 `VITE_API_URL`

### 重新构建
```bash
pnpm run clean
pnpm run setup
```
